# VK-TG-MESSAGES-CONVERTER-BOT
Бот для пересылки сообщений из ВК и Телеграм и наоборот
# Инструкция по установке
- создаёт бота в телеграме через @BotFather, даём разрешение на чтение сообщений в чатах и берём его токен
- создаём сообщество в вк и получаем его токен, в настройках callback api ставим версию 5.126 и вписываем ip или домен хоста где лежит бот
- полученые токены вписываем в config.php
- **не забываем про токен пользователя вк, лечше создать отделюную страничку ведь загруженые файлы могут напрягать**
- создаём standlone приложение и получаем на его id свой access токен(ссылка в файле конфига)
# Инструкция по использованию
Для того чтобы установить связь нужно узнать свой ID телаграма:
- /getMyID в лс бота телеграм
- Далее нужно добавить бота в беледу и дать ему права админа(без них бот сможет только пересылать сообщения из вк в телеграм)
- Далее прописывает в беседе с ботом
/invile АЙДИ_ТЕЛЕГРАМА
- Бот вернёт запись вида 20000... это код беседы
- Далее пишем боту в тегергам /chats и он вернёт список чатов в которые вас пригласили
- Пример:
- Ваши чаты:
- 21

отсюда нам понадобится число слева(21 в моём случае)
Создаём чат и добавляем туда бота
Пишем /set ЧИСЛО_КОТОРОЕ_МЫ_ПОЛУЧИЛИ
